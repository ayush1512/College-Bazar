## Setup
- Clone the project locally.

Using CLI
  ```bash
  gh repo clone ayush1512/College-Bazar
  ```
OR

Download the .zip file and extract.

- Open terminal.
  ```bash
  cd ../College-Bazar
  ```

- Using npm.
  ```bash
  npm i
  ```

- Using node.
  ```bash
  node app.js
  ```
